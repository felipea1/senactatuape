ctrl + l (limpar a tela)

sudo na frente para atualizar

Sudo apt update <-- atualizar as lista do apt (atualização de softwares)

sudo apt upgrade <-- atualiza todos os softwares

sudo apt full upgrade <-- atualiza todos os softwares (atualização completa)

sudo apt dist-upgrade <-- atualiza o sistema operacional (Kernel) 

sudo apt autoremove <-- remoção de softwares não mais nescessários

sudo apt autoclean <-- fazer a limpeza no sistema

sudo reboot <-- reiniciar 

sudo apt install <-- instalar aplicativos 

sudo apt install psensor
